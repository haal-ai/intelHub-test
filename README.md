# intelHub-test

**Test / Demo Technology Radar** — sample data for IntelHub development and testing.

## Purpose

This hub contains sample technology radar data used for developing and testing the IntelHub desktop app and MCP server. It is NOT a production intelligence hub.

## What belongs here

- Sample intells across all four quadrants (tools, process, technology, other) and all rings
- Test data for validating IntelHub features (snapshots, exports, search, etc.)

## What does NOT belong here

- Real intelligence items (use `inovhub` for innovation pipeline or `intelhub-gee` for tech radar)
- Production data of any kind

## Hub role

`team` — test/demo hub with sample data.
